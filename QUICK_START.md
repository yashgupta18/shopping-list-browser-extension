# Quick Start Guide

## 🚀 Get Started in 3 Steps

### Step 1: Generate Icons

1. Open `generate-icons.html` in your browser
2. Click "Generate All Icons"
3. Download all three icons (icon16.png, icon48.png, icon128.png)
4. Save them in the `icons/` folder

### Step 2: Install Extension

1. Open Chrome/Edge and go to `chrome://extensions/`
2. Enable "Developer mode" (toggle in top right)
3. Click "Load unpacked"
4. Select the `shopping-extension` folder

### Step 3: Start Using

1. Visit any supported site (see list below)
2. Go to your wishlist or cart page
3. Click the extension icon in your toolbar
4. See all your items in one place!

## 📱 Supported Sites (8 Indian E-commerce Platforms)

- **Ajio**: ajio.com
- **Flipkart**: flipkart.com
- **Amazon India**: amazon.in
- **Myntra**: myntra.com
- **Snitch**: snitch.com
- **Lifestyle Stores**: lifestylestores.com
- **Uniqlo**: uniqlo.com
- **Shoppers Stop**: shoppersstop.com

## 💡 Tips

- The extension automatically detects items when you visit wishlist/cart pages
- Click "Refresh" in the extension to reload items
- Use filters to view items by site or type
- Click "View on [Site]" to open items on the original website
