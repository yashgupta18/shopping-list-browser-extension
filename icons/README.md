# Icons Required

This extension requires three icon files to function properly:

- **icon16.png** (16x16 pixels) - For the extension toolbar
- **icon48.png** (48x48 pixels) - For the extension management page
- **icon128.png** (128x128 pixels) - For the Chrome Web Store and installation

## How to Create Icons

### Option 1: Use an Online Icon Generator

1. Visit a free icon generator like:
   - https://iconifier.net/
   - https://favicon.io/
   - https://www.canva.com/

2. Create or choose a shopping-related icon (shopping bag, cart, box, etc.)

3. Download in PNG format at the required sizes

4. Rename the files to match the required names and place them in this folder

### Option 2: Create Simple Icons with Any Image Editor

1. Open any image editor (Paint, Preview, Photoshop, GIMP, etc.)
2. Create a new image with the required dimensions
3. Draw or paste a simple shopping-related symbol
4. Save as PNG with the required filenames

### Option 3: Use Free Icon Resources

1. Download free icons from:
   - https://icons8.com/
   - https://www.flaticon.com/
   - https://iconmonstr.com/

2. Resize to the required dimensions
3. Save with the correct filenames

## Icon Suggestions

For a shopping extension, consider these icon styles:

- Shopping bag
- Shopping cart
- Gift box
- Package/parcel
- Price tag
- Wishlist heart

## Quick Start

If you just want to test the extension, create three simple square PNG files with:

- A solid color background
- Text "16", "48", or "128" in the center
- Save as icon16.png, icon48.png, and icon128.png

This will allow the extension to load and function for testing purposes.
